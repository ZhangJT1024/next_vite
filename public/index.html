📁 项目结构

vite/
├── src/
│   ├── api/          # API 接口
│   ├── components/   # 公共组件
│   ├── layouts/      # 布局组件
│   ├── router/       # 路由配置
│   ├── store/        # Pinia 状态管理
│   ├── styles/       # 全局样式
│   ├── types/        # TypeScript 类型
│   ├── utils/        # 工具函数
│   ├── views/        # 页面组件
│   │   ├── dashboard/   # 仪表盘
│   │   ├── login/       # 登录页
│   │   ├── register/    # 注册页
│   │   ├── user/        # 用户管理
│   │   ├── role/        # 角色管理
│   │   └── settings/    # 系统设置
│   ├── App.vue          # 根组件
│   └── main.ts          # 入口文件
├── package.json
├── vite.config.ts
└── README.md

🎯 功能模块

  • 仪表盘 (Dashboard) - 数据统计、图表展示、最近操作
  • 用户管理 - 用户列表查看、编辑、状态管理
  • 角色管理 - 权限码配置、角色启用/禁用
  • 系统设置 - 基本信息、安全设置、数据备份
  • 登录/注册 - 用户名密码登录、用户注册
🚪 进入登录页面 📝 进入注册页面 📊 直接进入仪表盘

💻 技术栈

Vue 3 Vite 5.2 Element Plus Pinia TypeScript Vue Router

📖 使用说明

本地开发： npm run dev
构建生产版本： npm run build
预览构建结果： npm run preview
代码检查： npm run lint
